Створення багатосторінкового сайту з використанням основних тегів HTML
Автор: bohdanavol

Мета: cтворити невеликий багатосторінковий сайт з коректною семантикою та внутрішньою навігацією, 
використовуючи лише HTML. Закріпити роботу з основними тегами та структурою документа, підготувати 
сторінки до базових SEO-вимог. 

Структура сайту:
1) index.html — Головна 
  ● Коротке привітання у <p>. 
  ● Одне зображення (логотип/ілюстрація) у <figure> з підписом у <figcaption>. 
  ● Список (ul або ol) з 3–5 пунктів про розділи сайту. 
  ● На цій сторінці використати один із заголовків h2–h6 (наприклад, підзаголовок розділу).
2) about.html – Про автора.
  ● Опис у 2–3 абзаци (<p>), використати логічні виділення: em, strong, а також хоч по одному з i, b, u 
(доречно й помірно). 
  ● Список захоплень/навичок: ul або ol. 
  ● Додатковий блок у <aside> (цитата/примітка), де використати <span> для короткого акценту. 
  ● Використати заголовки h2 і h3 для структурування. ry.html – Галерея.
3) gallery.html — Галерея 
  ● Щонайменше три елементи <figure> з <img> і підписом у <figcaption> (змістовний alt у зображень 
— обов’язково). 
  ● Опис категорій зображень у вигляді списку (ul/ol). 
  ● Додаткові підзаголовки рівнів h2–h4.
4) contacts.html – Контакти.
  ● Заголовок сторінки — h1 «Контакти». 
  ● Абзац із поясненням способів зв’язку. 
  ● Список контактів (ul або ol) з посиланнями mailto: і/або на соцмережі. 
  ● Один додатковий підзаголовок (h2 або h3). 
Папки:
- images – зберігає фото
